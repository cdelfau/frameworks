Typical Typography

One of the things bootstrap does is define CSS for certain HTML tags, such as heading styles:

H1 style! Hello, world! Small H1 style!

H2 style! Hello, world! Small H2 style!

H3 style! Hello, world! Small H3 style!

H4 style! Hello, world! Small H4 style!

. . . up to h7!

These are normal heading styles. Use <h1>, <h2>, <h3>,... etc. To get the subheadings, use <small> within a heading style pair.

Bunches of Buttons

... and buttons?!?!?!?...










vs.



You can mix and match different colors, sizes and states for your buttons by putting their codes isnide the class attribute of the button tag. For example:
<button class="btn btn-danger btn-xs active">

Color Size State
btn-default Default btn-lg Large active Active
btn-primary Primary btn-sm Small disabled="disabled" Disabled **
btn-success Success btn-xs X-small
btn-info Info btn-block Block *
btn-warning Warning
btn-danger Danger
* block size will make the button expand to the width of its container
** EXCEPTION: disabled is a separate attribute of the button tag -- should not go inside class attribute!
